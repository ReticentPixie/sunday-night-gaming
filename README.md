# Sunday Night Gaming
A simple web app to support the planning & logistics of a weekly game night among members of a private Discord server.

# Technologies Used
* HTML
* CSS
* JavaScript
* jQuery
* Google Fonts
* Font Awesome
* Giant Bomb API
